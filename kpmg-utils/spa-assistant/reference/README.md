# Reference Index

This folder stores SPA analysis inputs and outputs used by the assistant.

## Structure

- `source-documents/`
  - Original source agreement files (`.pdf`, `.docx`, and other raw documents).
- `analyses/`
  - Extracted and authored analysis outputs for SPA cases and edge-case handling.
- `playbooks/`
  - Modular playbook templates used for structured output.
- `support/`
  - Process notes, checklists, scoring rubrics, and prompting helpers.
- `datasets/`
  - Example corpora and schema files (`*.csv`, `schema.json`, `term-map.yml`).
- `research/`
  - Research prompts and investigation notes.
- `prompts/`
  - Prompt pack used for extraction workflows.
- `consolidated/`
  - Consolidated artifacts compiled across multiple analyses.

## Notes

- The folder is intentionally separated from `docs/` so that source material can be reused, audited, and replaced independently from product specification.
