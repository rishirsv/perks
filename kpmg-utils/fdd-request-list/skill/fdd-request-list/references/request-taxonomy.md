# Request Taxonomy

Use this taxonomy to build the request-list spec before generating the workbook.

## Core FDD Backbone

Map the first-round list to these KPMG TS-aligned areas:
- Accounting overview
- Quality of earnings
- Revenue
- Operating expenses
- Working capital
- Accounts receivable
- Accounts payable and accrued liabilities
- Capital expenditure requirements
- Net debt
- Commitments and contingencies

## Core Workbook Modules

Use these section titles when the engagement needs the standard first-round backbone:

### Highest priority

Lead with the core unlock requests:
- Annual financial statements
- Monthly trial balances
- Consolidating financial statements when multiple entities exist
- Management add-backs and detailed support
- Transaction-level revenue or billings data
- Budget and forecast

### High priority

Add the next set of operational FDD requests:
- KPIs and operating metrics
- Headcount trending and employee census
- AR aging and reserve support
- Unbilled receivables
- Deferred revenue and waterfall support
- Management proposed NWC target and cycle commentary

### Medium priority

Use for supporting but non-gating asks:
- purchase journal
- AP aging

### Low priority

Use for broad catch-all balance sheet and legal-exposure asks:
- commitments and contingencies

## Conditional Modules

Add these modules only when they fit the deal facts:
- If sell-side report available
- If audited and workpaper access necessary
- If carve out
- If monthly reporting packages deemed high value source of information
- If discontinued operations or sun-setting product line
- If acquisitions during historical scope period
- If business has start-up operations
- If forecast is in scope, or deferred revenue and backlog material
- If currency exposure
- If manufacturing environment or standard costing
- If review of customer contracts critical to diligence
- If supplier concentration or rebate exposure
- If product or service pricing critical to diligence
- If software or subscription business
- If IT managed services business
- If professional services firm or project based work
- If inventory is key diligence area
- If sales channel information is critical
- If product mapping or hierarchy is not included in transaction data
- If capex or fixed assets are material
- If transaction data is unavailable
- If Target is in a highly regulated market
- If more than one legal entity or a complex JV structure exists
- If multiple locations or rent is material
- If royalties or license agreements exist
- If marketing expenses are material
- If related intercompany transactions are significant
- If related party transactions are significant
- If asset retirement obligations or environmental reserves are relevant
- If commissions are material
- If professional fees are material or fluctuate
- If Company is self insured
- If significant open positions exist
- If direct versus indirect labor split is unclear
- If retention compensation or change-of-control provisions exist
- If detailed testwork procedures are required
- If current debt structure is meaningful to the client
- If executive compensation programs are complex
- If public company becoming private post acquisition
- If cost savings initiatives or restructuring plans are underway
- Other balance sheet requests
- Other potential items to consider if material
- If Energy deal
- If Midstream energy deal
- If Regulated Utility deal

## Writing Rules

- Keep request wording direct and Excel-friendly.
- Preserve standard request phrasing from the reference workbook where possible.
- Add KPMG comments only when they improve usability, such as reconciliation instructions, bridge requirements, or known focus areas.
- Keep the first round focused on requests that unlock analysis, not every imaginable diligence question.
