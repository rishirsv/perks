# ISSUES.md

## Purpose
Single chronological issue registry with severity markers.

## Severity Scale
- `[P0]` Critical: production blocker/data-loss/security-critical.
- `[P1]` High: major user-facing breakage or release-gating regression.
- `[P2]` Medium: meaningful defect with workaround.
- `[P3]` Low: minor bug, polish, or non-blocking drift.
