# Tech Debt Tracker

## Scoring Rubric
Score each item as:
- impact: 1-5
- frequency: 1-5
- effort: 1-5
- priority score: `(impact * frequency) / effort`

## Debt Garbage Collection Cadence
- Review cadence: every 30 days.
- During each review: refresh owner, next step, and linked artifact.
