# Rhythm and visual storytelling

This is for page/flow cohesion and “reading order.”

## Reading order
Each section should have:
- a clear headline
- one sentence of “why it matters”
- proof/details (optional)
- a clear action or next step (optional)

## Density and pacing
- Alternate dense and breathable regions to reduce fatigue.
- Use consistent section spacing and predictable transitions.

## Grouping and scannability
- Prefer whitespace and alignment over extra borders.
- Use subheads to chunk content and create scan anchors.

## System consistency
- Buttons, cards, and inputs should behave like a system:
  same radii, same padding families, same focus treatment.
