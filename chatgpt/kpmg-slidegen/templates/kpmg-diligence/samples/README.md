# samples/

This folder holds **input JSON** used to generate demo decks.

- `demo.json` — small, generic example used by `npm run gen:deck`.
- `v1-nvidia-v2.json` — the current “latest NVIDIA” sample input used by the generator.
- `v1/` — legacy inputs kept only for migration tools.

Generated artifacts (PPTX/PDF/PNGs) should go to `outputs/` (throwaway, gitignored).
