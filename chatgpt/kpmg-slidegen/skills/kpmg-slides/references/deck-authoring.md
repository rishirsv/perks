# Deck Authoring Reference

Use this reference to produce client-style, high-density diligence narratives.

## Narrative architecture

1. Open with context and decision objective.
2. Present evidence with explicit analytical interpretation.
3. Close each section with implications and action-oriented next steps.
4. Maintain logical progression from findings to decision impact.

## Layout selection guidance

1. Use `oneColumnText` for unified narratives and synthesis.
2. Use `twoColumnText` for compare/contrast or dual-thread analysis.
3. Use `analysisNarrowTable` for metric-heavy snapshots and compact tabular evidence, and always include authored `insights` bullets.
4. Use `analysisWideChart2ColsText` when narrative and chart evidence should be read together.
5. Use `analysisWideChartTableText` for integrated chart + table + interpretation on one page.
6. Use `titleStrapline4TextBoxes` for workstream or pillar-based execution plans.

## Density and readability

1. Keep paragraphs complete, precise, and analytical rather than generic.
2. Use subheaders to segment dense sections and improve executive scanning.
3. Use bullets for action lists, risk registers, and concise implications.
4. Avoid decorative text that does not affect decision quality.
5. Include source-style notes on slides that present quantitative evidence.

## Writing standards

1. Write in executive tone with complete sentences and explicit claims.
2. Anchor statements with quantified facts where possible.
3. State caveats and uncertainty directly rather than implying certainty.
4. Keep wording commercially realistic and transaction-aware.
5. Ensure each slide has a clear take-away that supports a decision.

## Revision loop

1. Read QA first and identify blocking issues before cosmetic edits.
2. Fix required-slot and overflow drivers before style refinement.
3. Regenerate after each substantive change and compare outputs.
4. Track whether issues are structural, data-shape, or content-density related.
5. End with a concise summary of what changed and what remains.
